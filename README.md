Python FastAPI app that currently allows you to download spotify and youtube playlists (downloads best quality OPUS from youtube).

Frontend utilises Jinja, HTMX, Alpine.js and Tailwind.

blah blah blah, still in progress, etc etc.

# Requirements:
### Python
- dotenv
- fastapi
- socketio
- sqlalchemy
- jinja
- yt_dlp
- spotipy
- ytmusicapi
- mutagen (pretty sure this is a requirement for yt_dlp to embed opus metadata)

### Other
- tailwind
- a database
